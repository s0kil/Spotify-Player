# Spotify Web Player

## Status

Project paused until Spotify allows to use the Web Playback SDK for non-premium users.

Ref: https://developer.spotify.com/documentation/web-playback-sdk/#requirements

## Get started

Install Dependencies
`npm install`

Start Parcel
`npm run dev`

## Build Production

`npm run build`
